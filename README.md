# MSX-PLAYer-GCA-Reader
MSX-PLAYer Game Cartridge Adapter （以下、MGCAと略します）用の、カートリッジを読み取ってファイルに保存するダンプツールです。

とりあえず、なんとか動くところまで作ってみました。

.NET 5.0 (x86)のランタイムが必要です。
下記のMicrosoft公式サイトのURLから、「.NET デスクトップ ランタイム 5.0.17」の「x86」版をダウンロードして、インストールしてください。
https://dotnet.microsoft.com/ja-jp/download/dotnet/5.0

## 使い方
exeを起動すると、USBシリアルポートでMGCAを検出して、自動で割り当てられている「COMxx」が選択されます。

複数の機器をつないだ場合は、使用するアダプタを選択可能です。（多分、、、2台ないので未テスト）

搭載されているスロットの数を検出して、使用するスロットを選択可能です。通常は1が選択されます。
（こちらも、2スロットタイプがまだないので未テスト）

ツール起動後にMGCAをPCに接続した場合は、「ポート再読込」ボタンを押すと認識されます。

「ダンプ開始」ボタンを押すと、保存するファイル名の指定になるので、保存先とファイル名を指定してください。
既に存在するファイルを指定した場合は、上書きするかどうかを聞かれます。

メガロムの場合、「中止」ボタンを押すとダンプを中断可能です。

以下のROMタイプに対応しました。タイプの自動検出はまだできませんので、手動で選択してください。

- NORMAL(8K/16K/32K)
- ASCII 8K
- ASCII 16K
- KONAMI
- KONAMI SCC
