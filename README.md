# MSX-PLAYer-GCA-Reader
MSX-PLAYer Game Cartridge Adapter （以下、MGCAと略します）用の、カートリッジを読み取ってファイルに保存するダンプツールです。

<img width="621" height="428" alt="MGCAR-0 3 0" src="https://github.com/user-attachments/assets/11a53949-0f10-4b51-8f09-8ea453c17ff4" />

.NET 10.0 (x86)のランタイムが必要です。

下記のMicrosoft公式サイトのURLから、「.NET デスクトップ ランタイム 10.0.8」の「x86」版をダウンロードして、インストールしてください。
https://dotnet.microsoft.com/ja-jp/download/dotnet/10.0

## 使い方
exeを起動すると、USBシリアルポートでMGCAを検出して、自動で割り当てられている「COMxx」が選択されます。

複数の機器をつないだ場合は、使用するアダプタを選択可能です。（多分、、、2台ないので未テスト）

搭載されているスロットの数を検出して、使用するスロットを選択可能です。通常は1が選択されます。
（こちらも、2スロットタイプがまだないので未テスト）

ツール起動後にMGCAをPCに接続した場合は、「ポート再読込」ボタンを押すと認識されます。

ROMタイプを選択して、「ダンプ開始」ボタンを押すと、保存するファイル名の指定になるので、保存先とファイル名を指定してください。
既に存在するファイルを指定した場合は、上書きするかどうかを聞かれます。

メガロムの場合、「中止」ボタンを押すとダンプを中断可能です。

以下のROMタイプに対応しました。任意のROMタイプを選択してください。

- (Auto) ASCII 8K/16K、KONAMI/SCC/NORMALを自動識別する
- NORMAL(8K/16K/32K)
- NORMAL 48K
- NORMAL 64K
- ASCII 8K
- ASCII 16K
- KONAMI
- KONAMI SCC
- R-TYPE
- CROSS BLAIM
- HALNOTE/日本語標準カートリッジ

## 自動判別
「個別自動判別」にチェックしておくと、以下のどれかかどうかをまず判定します。
（最初の16KBのチェックサムで判定しているため、バージョン違いは検出できない可能性あり）
- R-TYPE
- CROSS BLAIM
- HALNOTE/日本語標準カートリッジ
